# ConiCheesecake

Sitio web estático para una página de tartas de queso con un diseño cálido, moderno y orientado a pedidos por WhatsApp.

## Estructura de archivos

- `index.html` - página principal con secciones tipo ventanas.
- `css/styles.css` - diseño premium, paleta cálida y responsive.
- `js/script.js` - menú móvil.

## Diseño actualizado

- Hero con fondo de cheesecake, llamada a la acción y mini ventanas destacadas.
- Secciones separadas como ventanas para tartas, precios, historia y contacto.
- Apariencia más cálida y apetecible para inspirar encargos.

## Cómo funciona

1. `index.html` contiene las secciones:
   - Tartas
   - Ventanas informativas
   - Precios
   - Sobre mí
   - Contacto
2. El botón principal abre WhatsApp con un mensaje de encargo.
3. No hay tienda online; los pedidos se realizan por WhatsApp.

## Personalizar

- Cambia el número de WhatsApp en `index.html`.
- Ajusta los sabores, precios y descripciones.
- Si quieres usar imágenes locales, crea una carpeta `assets/` y reemplaza las URLs de imagen en `index.html`.

## Ver la web

Abre `index.html` en el navegador o usa un servidor local si quieres probar la navegación.
